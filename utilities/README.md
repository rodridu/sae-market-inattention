# Utility Scripts

Helper scripts for validation, diagnostics, and maintenance.

## Validation

- `validate_data.py` - Validate input data before pipeline
- `validate_pipeline.py` - Comprehensive pipeline status check
- `verify_alignment.py` - Verify embedding-sentence alignment

## Diagnostics

- `diagnose_merge.py` - Debug accession number format issues
- `monitor_training_loss.py` - Real-time SAE training loss monitoring

## Maintenance

- `fix_cik_and_resample.py` - Add CIK to sentences and re-sample
- `merge_wrds_data.py` - Merge WRDS controls with outcomes
- `stratified_sampling.py` - Stratified sampling utilities
- `show_stratification_stats.py` - Display sample statistics
